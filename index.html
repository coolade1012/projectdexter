About This Project

Project Dexter is a one person project utilizing multiple technogolies to create an online Pokédex. A Pokédex is an encyclopedia of all the creatures in Pokémon. Unfortunately, the Pokédex within the games is incomplete, thus players refer to websites that offer the additional information in a table or spreadsheet-like format. The user experience when using these Pokédex websites is neither interactive nor visually appealing.

This projects goal is to create an interactive, intuitive, and visually appealing website with an underlying database that works on all browsers and is meant to be just an extension of the Pokémon video games.
