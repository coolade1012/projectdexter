About Us

A Pokédex is an encyclopedia of all the creatures in Pokémon, the collection of anime characters in the video and trading card games first released in 1996. Unfortunately, the Pokédex within the games is incomplete, thus players refer to websites that offer the additional information in a table or spreadsheet-like format. The user experience when using these Pokédex websites is neither interactive nor visually appealing.

This project created an interactive, intuitively usable, and visually appealing website with an underlying database that feels like an extension of the Pokémon game.
